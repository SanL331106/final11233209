產品設計

理性與感性的融合

蛋盒設計

作品名稱 : 蛋生Born ； 課程 : 大一造形基礎

專題目標 : 透過規定的基本材料(牛皮紙)學習對立體造型與紙模製作的掌握及熟練度，並在禁止用膠的限制下，思考如何設計出牢固且具創意性的結構與卡榫。

設計理念 : 以鳥巢為造形概念，創作出一個富有生命力的蛋盒。

小北百貨的燈具設計

作品名稱 : 蒲公英燈罩 ； 課程 : 大一造形基礎

專題目標 : 從生活百貨中找出適合的材料，以數個單元式的小結構重複排列組裝構成兼具理性與美感的創意燈罩。

設計理念 : 以蒲公英為造形概念，結合暖光燈泡，製作出有如家一般溫暖的燈具。

雷射切割創意商品開發

作品名稱 : 海風-水母風鈴 ； 課程 : 大二產品設計

專題目標 : 以小組為單位，針對所選的設計標地設計出系列產品，以造型美感為重，並學習雷射切割機器操作以及卡榫結構設計。

設計理念 : 融合水母的透明感以及風鈴的靈動，將水母觸手化作風鈴掛墜，設計出一款精緻且靈氣的風鈴。

不插電生活用品設計

作品名稱 : 樹葉拉繩水龍頭 ； 課程 : 大二產品設計

專題目標 : 以小組為單位，從品牌概念發想到提案再到製作，完整走過產品設計流程，並學習3D列印機器操作以及模型打磨、補土與噴漆等技能。

設計理念 : 透過柔和的曲面設計和鮮豔的色彩，結合樹葉造型與小瓢蟲以及拉繩玩具的特色，打造一個可愛又實用的水龍頭。
